# set-out.com

This is the official website for SET-OUT INTERNATIONAL COMPANY LIMITED, a company specializing in technology services, import/export, consultancy, and training.

## About

SET-OUT INTERNATIONAL COMPANY LIMITED provides comprehensive business solutions including:
- Import/export and commerce of hardware and software
- Consultancy in electrical engineering, IT, and communication
- IT subcontracting and project management
- Training and professional development
- Digital marketing and software development
- Technical assistance

## Tech Stack

- **Framework**: Next.js 15 with React 19
- **Styling**: Tailwind CSS with shadcn/ui components
- **Language Support**: Multi-language (English, French, Portuguese, Traditional Chinese, Simplified Chinese)
- **TypeScript**: Full type safety enabled

## Getting Started

1. Install dependencies:
   ```bash
   cd my-app
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Project Structure

- `app/`: Next.js app router pages and layouts
- `components/`: Reusable React components
- `contexts/`: React context providers (language support)
- `lib/`: Utility functions and translations
- `public/`: Static assets
